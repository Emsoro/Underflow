## ✨ 为什么我这么爱它

### 1️⃣ 打开就能用，完全免费

❌ 不需要注册
❌ 不需要登录
❌ 没有任何限制

双击 exe 文件就打开了，就这么简单。

### 2️⃣ 操作超级简单

顶部就三个按钮：
♡ Start → 流程起点
♤ Node → 处理节点
♧ End → 流程终点

点一下添加节点，拖一下把手就连线
3 分钟就能画出一张流程图 🎯

### 3️⃣ 颜值在线 💅

给节点配色超方便！
两排色块直接点选：
- 填充色：节点整体变色
- 边框色：只改边框颜色

还有自定义取色器，想要什么颜色都有

一键切换实线/虚线边框
流程图瞬间层次感拉满 ✨

### 4️⃣ 节点形状随意换

同一个节点可以在 **矩形** 和 **菱形** 之间切换
不用一开始就想好要用什么形状
画到一半随时改，超灵活

### 5️⃣ 双击就能改文字

双击节点改文字
双击连线改标签
不用找编辑按钮，超直观

### 6️⃣ 导出无水印

✅ SVG 矢量图（插 Word 超清晰）
✅ PNG 图片（直接发微信/钉钉）
✅ JSON 工程文件（随时打开继续改）

没有任何水印，完全免费！

### 7️⃣ 文件超小

安装包不到 10MB！
启动秒开，不卡不占内存
比我之前用的 Electron 工具轻多了
