## 一句话介绍

**UnderFlow 是一款基于 React Flow + Tauri 构建的轻量级流程图桌面应用，主打本地运行、操作简单、导出无限制。**

## 核心功能一览

### 📐 节点创建

顶部工具栏三个按钮即可开始绘图：

- **♡ Start** —— 流程起点（输入节点）
- **♤ Node** —— 处理节点（矩形，默认形状）
- **♧ End** —— 流程终点（输出节点）

点击按钮即可在画布上添加节点，拖拽调整位置，拖拽节点把手即可连线。

### 🔷 节点形状切换

选中节点后，右侧属性面板的 **Shape** 下拉框可以在 **矩形** 和 **菱形** 之间切换：
- 矩形：适合流程步骤、处理环节
- 菱形：适合条件判断、分支决策

无需预先决定节点类型，画到一半随时切换。

### 🎨 丰富的配色方案

属性面板提供两排独立的色块选择器：

| 色块行 | 类型 | 效果 |
|--------|------|------|
| **Fill（填充）** | 实心圆 | 节点整体着色，文字自动变白 |
| **Border（边框）** | 空心环 | 仅改变边框颜色 |

每行 9 个预设色块（无 / Rose / Coral / Amber / Emerald / Cyan / Blue / Violet / Pink），外加一个自定义取色器，点击渐变色轮即可选取任意颜色。

### 📏 边框样式

- **Solid（实线）** —— 默认样式
- **Dashed（虚线）** —— 适合标注辅助节点或可选流程

### ✏️ 内联编辑

- **双击节点** —— 直接编辑节点文字
- **双击连线** —— 编辑连线标签
- 编辑完成后点击空白处或按 Enter 确认

### 📤 导出与保存

| 格式 | 说明 |
|------|------|
| **SVG** | 矢量图，适合插入文档、印刷 |
| **PNG** | 位图，适合截图分享 |
| **JSON** | 完整工程文件，可随时打开继续编辑 |

### 🔄 自动布局

点击工具栏的 **⟳ Layout** 按钮，elkjs 引擎自动整理节点位置，一键对齐，告别手动拖拽的繁琐。

### 📦 可折叠属性面板

右侧属性面板通过一个精致的标签按钮收起/展开：
- 收起后画布全屏展示
- 展开后即时编辑节点属性
- 过渡动画流畅自然

## 快速上手指南

### 步骤 1：添加起始节点

点击 **♡ Start** 按钮，画布上出现一个起始节点，默认文字为 "Start"。

### 步骤 2：添加处理节点

点击 **♤ Node** 按钮，在适当位置放置处理节点。

### 步骤 3：连线

将鼠标悬停在节点的圆形把手上，拖拽到另一个节点的把手上松开，即可创建连线。

### 步骤 4：添加条件分支

双击处理节点打开属性面板，将 Shape 改为 **Diamond**，节点变为菱形，适合做判断节点。

### 步骤 5：美化样式

在属性面板中为不同节点设置填充色和边框色，区分不同流程阶段。

### 步骤 6：导出成果

点击 **↓ SVG** 或 **↓ PNG** 导出图片，或点击 **💾 Save** 保存为 JSON 文件。

## 与其他工具的差异

### vs Draw.io

| 维度 | Draw.io | UnderFlow |
|------|---------|-----------|
| 安装包大小 | ~100MB (Electron) | < 10MB (Tauri) |
| 节点库 | 200+ 种 | 聚焦核心（矩形/菱形） |
| 数据位置 | 可选云端 | 完全本地 |
| 学习曲线 | 功能多需熟悉 | 3 分钟上手 |

**选择建议**：需要画 UML/ER/网络图选 Draw.io；只需要画流程图选 UnderFlow。

### vs ProcessOn

| 维度 | ProcessOn | UnderFlow |
|------|-----------|-----------|
| 使用方式 | 在线平台 | 本地应用 |
| 是否需要注册 | 是 | 否 |
| 免费版限制 | 有文件数量限制 | 无限制 |
| 数据隐私 | 上传到云端 | 完全本地 |

**选择建议**：喜欢模板和团队协作选 ProcessOn；注重隐私和效率选 UnderFlow。

## 技术亮点

对于技术爱好者，UnderFlow 的技术栈也值得一看：

- **前端**：Vite + React + TypeScript，现代化开发体验
- **流程图引擎**：@xyflow/react（React Flow v12），业内最流行的 React 流程图库
- **自动布局**：elkjs，Eclipse Layout Kernel 的 JS 实现
- **桌面框架**：Tauri v2，Rust + WebView2，比 Electron 小巧 10 倍
- **文件操作**：Rust 原生 fs API + rfd 对话框，性能出色
